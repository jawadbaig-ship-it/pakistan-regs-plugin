# UN Security Council Sanctions - Pakistan Implementation

**Regulator:** GoP / MoF  
**Reference:** UN Sanctions Implementation  
**Date:** Live  
**Applicable to:** All Reporting Entities  
**Source:** [https://www.finance.gov.pk/](https://www.finance.gov.pk/)

## Purpose & Scope

Pakistans implementation of UNSC sanctions regimes (1267, 1373, 1718, etc.) - SROs and lists are issued by MoF; FIs must screen against these in real time.

## Key Sections / Requirements

- UNSC 1267 (Al-Qaida/ISIL/Taliban)
- UNSC 1373 (terrorism financing)
- UNSC 1718/2231 (DPRK/Iran)
- MoF SROs and notifications
- FATF Schedule II/IV (Pakistan-specific)
