# Open Banking Framework (Evolving)

**Regulator:** SBP / SECP  
**Reference:** SBP DFS - Open Banking  
**Date:** Evolving  
**Applicable to:** Banks, MFBs, Digital Banks, EMIs, fintech aggregators  
**Source:** [https://www.sbp.org.pk/dfs/index.htm](https://www.sbp.org.pk/dfs/index.htm)

## Purpose & Scope

Forward-looking SBP-led framework for account aggregation, payment initiation and standardised APIs across banks/EMIs/digital banks.

## Key Sections / Requirements

- API standards (auth, KYC, account, payments)
- Consent-based data sharing
- Liability allocation between participants
- Security and resilience
- Customer disclosures and dispute handling
- Phased rollout - aggregation, AISP, PISP
