# Suspicious Transaction Report (STR)

*STR*

## Definition

Mandatory report filed with FMU when a Reporting Entity suspects a transaction relates to ML/TF/predicate offences. No threshold — based on suspicion. Tipping-off the customer is criminalised.

## Why this matters

Filed via goAML portal. Failure to file is an offence under AML Act 2010.
